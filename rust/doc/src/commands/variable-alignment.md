# VARIABLE ALIGNMENT

`VARIABLE ALIGNMENT` sets the alignment of variables for display
editing purposes.  It does not affect the display of variables in PSPP
output.

```
VARIABLE ALIGNMENT
        VAR_LIST ( LEFT | RIGHT | CENTER )
        [ /VAR_LIST ( LEFT | RIGHT | CENTER ) ]
        .
        .
        .
        [ /VAR_LIST ( LEFT | RIGHT | CENTER ) ]

```
